## Installation

App membutuhkan [Node.js](https://nodejs.org/) v16+ untuk dapat berjalan.

For Development

```sh
cd finalProject2
npm install
npm run develop
```

For Production

```sh
cd finalProject2
npm install
npm run build
npm run start
```
